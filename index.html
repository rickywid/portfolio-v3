Movies Review

Ruby on Rails

For this project I used several different gems to put the site together. Devise was used for user authentication/authorization, Simple Forms to create the forms, Paperclip to allow file attachments along with Bootstrap and HAML. There were a few challenges when trying to deploy the site to Heroku to make certain features work.

First, Heroku doesn't allow data storage on their servers for the form image attachments(for free account users as far as I know) so I signed up for Amazon's 1 year free AWS S3 cloud storage to allow the file attachments to work. After installing the necessary gems and some configuration, I was able to get everything working.

Secondly, I had used the ElasticSearch gem for the search functionality. I got the search feature to work locally but once I deployed to Heroku it was failing. After days of troubleshooting, I eventually found out that in order to use ElasticSearch on Heorku, you needed to pay a fee. So eventually I had to remove the search functionality although locally everything was working great.

Overall, I learned a lot about Rails and I feel I made great strides by completing this project.
