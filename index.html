Movies Review

Ruby on Rails

For this project I used several different gems to put the site together. Devise was used for user authentication/authorization, Simple Forms to create the forms, Paperclip to allow file attachments along with bootstrap and haml. Getting search certain features to work on Heroku turned out to be troublesome.

First, Heroku doesn't allow data storage on their servers (for free account users as far as I know) so I signed up for Amazon's AWS S3 cloud storage to allow the file attachments to work. This worked perfectly even though I spent quite some time to configure Amazon S3 to cooperate with Heroku.

Second, I had originally had a search functionality to allow users to search movies on the site. ElasticSearch was used for this feature. Locally everthing was working great but once I deployed to Heroku it suddenly stopped. I spent days trying to figure it out. After finally figuring out what the issue was, I realized that you needed to pay in order to use ElasticSearch with Heroku's services. So unfortunately I had to remove the search function (works locally only).

Overall, I learned a lot about Rails and I feel I made great strides by completing this project.
